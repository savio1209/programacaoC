# Exercícios

Esta pasta reúne exercícios práticos de lógica de programação utilizando a linguagem C.

## Conteúdos abordados:
- Estruturas condicionais compostas
- Estruturas de repetição (`for`, `while`, `do-while`)
- Operadores lógicos
- Resolução de problemas práticos

## Objetivo:
Aprimorar o raciocínio lógico e a capacidade de resolver problemas através da programação.
