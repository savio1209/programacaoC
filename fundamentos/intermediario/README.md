# Intermediário

Exercícios de nível intermediário em C, focados em lógica mais estruturada e organização do código.

## Conteúdos abordados:
- Estruturas condicionais compostas (if/else if/else)
- Operadores lógicos (&&, ||)
- Estruturas de repetição (for, while, do-while)
- Uso de menus interativos (switch)
- Validação de entrada de dados

## Objetivo:
Desenvolver maior controle sobre o fluxo do programa e começar a resolver problemas mais próximos de situações reais.
