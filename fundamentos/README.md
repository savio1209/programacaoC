# Fundamentos

Esta pasta contém exercícios básicos da linguagem C, desenvolvidos durante o início dos estudos em programação.

## Conteúdos abordados:
- Variáveis e tipos de dados
- Entrada e saída de dados (`printf` e `scanf`)
- Estruturas condicionais simples (`if` e `else`)

## Objetivo:
Construir uma base sólida em lógica de programação e compreensão dos conceitos iniciais da linguagem C.
